Create a production ready React TypeScript frontend for a crypto strategy generation product called BestStrat.

BestStrat is a CoinMarketCap Strategy Skill product. It does not execute trades, does not connect wallets, and does not place live orders. It turns CoinMarketCap market data into a backtestable trading strategy spec.

Core product idea:
BestStrat helps users generate Quantopian style crypto strategies. The app detects market regime, analyzes market signals, generates entry rules, exit rules, risk rules, invalidation rules, and shows backtest results. The final output should be usable as a CMC Skill strategy spec.

Design style:
Dark professional crypto dashboard
Premium, serious, clean, data driven
Use deep black, charcoal, slate gray, white text, soft BNB yellow accents, green for positive results, red for risk warnings
No cartoon style
No meme style
No wallet connect
No login page
No fake trading execution buttons
No “Buy” or “Sell” action buttons
The app must clearly say it generates strategy specs, not financial advice

Code structure requirement:
Create clean React TypeScript code
Use Tailwind CSS
Use reusable components
Put API call logic in src/lib/api.ts
Put TypeScript response types in src/types/strategy.ts
Keep mock data separate in src/data/mockStrategy.ts
Do not hardcode final values inside components
Make it easy for a developer to replace mock data with the real backend API

Important:
The UI must feel like a CMC Skill strategy generation product, not a trading bot.

Market data input

Strategy generation
